# PVP Matching Web Client

This is the frontend client for the Bombcrypto PVP Matching system. It provides a web interface for managing and viewing PVP fixture matches, players, and test environments.

## Features
- View and manage PVP fixture matches and players
- Register new matches, players, and match groups
- Dashboard and test utilities for PVP features

## Getting Started

### Prerequisites
- Node.js (v16 or later recommended)

### Installation
1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm start
   ```

3. Open your browser and navigate to `http://localhost:3000` to use the app.

## Project Structure
- `src/` - Main source code
- `public/` - Static assets and HTML template
- `craco.config.js` - CRACO configuration for custom builds

## Build
To create a production build:
```sh
npm run build
```
