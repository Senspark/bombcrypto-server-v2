## 🚀 Global Patch Overview
This PR is part of the **Advanced Rarities Activation** patch, enabling Mega (6), Super Mega (7), Mystic (8), and Super Mystic (9) rarities across the BSC and Polygon networks. This patch expands the game's economy and loot pool, utilizing the pre-existing logic from the legacy Solana/TON implementations.

## 📦 Repository Specific Changes
Core modifications reside in `UserHeroFiManager.kt` and the accompanying database migration script `20260403_expand_rarities_6_to_9.sql`. 
- Overrode the legacy `generateWithoutNewRarity()` usage inside hero minting by replacing it with the full `generate()` function, thereby unlocking the extended loot table drops natively on the server. 
- Economically tuned the `config_min_stake_hero` and specific base settings with a +30% incremental tuning pattern for the new tiers. 
- Engineered Treasure Hunt reward arrays via Option B (Soft-cut) to smoothly redistribute rewards from pools 3, 4, and 5 sequentially into the high-end 6-9 pools, minimizing negative inflation on base players while aggressively rewarding high-tier collectors.

## ⚠️ Important Alerts & Economy Balance
- **+30% Growth Rule:** As per the new design, the `config_min_stake_hero` for the new rarities follows a +30% growth pattern to ensure a sustainable economy, rather than geometric doubling.
- **Isolaton:** Rarities 0-5 remain 100% unaffected.
